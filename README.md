# Details
This is a webcrawler built using python.You just have to give the url and the element and class of the website where the content is present.


# Installation

```sh
Install Python
$ pip install bs4
$ python webcrawler.py
enter the url
enter the element(eg: a,div) and the class 
enjoy 
```
